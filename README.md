# 前言/目录

## 为什么写学习笔记？

每次遇到不懂的技术性问题，总是去文档/Github/Google去找寻答案。虽然最终问题得到解决，但是没有一两次的查询，一个知识点是记不住的。时间浪费在了来来回回的查询上，所以就有了这个笔记！  
 这个计划在脑子里存在了很久，一直没有落实，主要原因还是拖延吧；说是拖延其实也不完全。虽然想记，但又感觉没有什么可以记的作为一个毕业不久的，项目经验啥的都没有，所以就拖到了现在。
